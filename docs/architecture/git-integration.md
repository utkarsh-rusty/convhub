# Git Integration

## Status

**Planned** (v1.2–v1.3) — not implemented.

## Intent

Git remains the source of truth for code. ConvHub will link project memory to repositories so developers can pull code and context together.

## Planned capabilities

- Git metadata on projects
- Repository linkage
- Push / pull context synchronized with Git workflows
- VS Code–assisted Git sync (v1.3)

## Not implemented

ConvHub does not currently read or write Git repositories.

See [roadmap.md](../../roadmap.md).
