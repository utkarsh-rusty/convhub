# Context Packages

## Status

**Planned** (v1.1) — not implemented.

## Intent

Context Packages will be portable bundles of project memory (conversations, commits, and related metadata) that can be published, shared, and restored across developers and tools.

## Planned capabilities

- Package conversations and commits into a portable unit
- Restore context into a workspace or IDE
- Decision tracking alongside packages

## Not implemented

No APIs, schema, or UI for Context Packages exist yet.

See [roadmap.md](../../roadmap.md) and [project-memory.md](project-memory.md).
