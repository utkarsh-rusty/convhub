# README screenshots

Drop PNG screenshots in this folder using these exact filenames so they appear on GitHub:

| File | Suggested content |
|------|-------------------|
| `landing.png` | Marketing / landing page |
| `conversation.png` | Shared conversation view |
| `branching.png` | Branch manager or branch tree |
| `commit-graph.png` | Commit graph |
| `dashboard.png` | App dashboard / home |
| `providers.png` | AI providers settings |

Tips:

- Prefer wide captures (about 1200–1600px wide).
- Use PNG or JPEG; update `README.md` paths if you change the extension.
- Paths in the root `README.md` are relative: `docs/images/<name>.png`.
